n.txt - содержит распределение групп по автобусам
p2.py - считывает распределение из n.txt распределение по автобусам (матриц N_mj, которая используется в штрафах),
создаёт соответствующую QUBO-матрицу и решает задачу QUBO. Далее получившийся бинарный вектор записывается в текстовый файл 
vector.txt
