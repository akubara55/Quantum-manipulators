1) Создание эмбендингов с помощью BertTokenizer, BertModel из библиотеки transformers. При этом строки отзывов преобразуются в вектора размерностью 768.
2) Вектора эмбендингов делились на train и test с помощью train_test_split
3) Количество положительных и отрицательных отзывов в датасете уравнивалось с помощью RandomOverSampler
4) К векторам эмбендингов применялась логистическая регрессия
5) На предсказанных с помощью логистической регрессией логитах происходило дообучение модели с помощью квантового алгоритма VQC с помощью которого удалось увеличить accuracy на тестовой выборке до 0.857
6) Был создан отдельно файл pyideem_qasm.py для симуляции классификации на квантовом компьютере,в который задаётся тестовая выборка и обученная модель.
Далее с помощью квантовой схемы из файла  svm.qasm и схемы ftest_1.qasm происходит классификация.
